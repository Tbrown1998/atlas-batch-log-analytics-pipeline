# Troubleshooting & Failure Scenarios

Atlas is designed to fail clearly.

---

## Common Failures

- Missing partitions
- Invalid input data
- Schema drift

---

## Recovery Strategy

1. Fix upstream issue
2. Re-run affected date
3. Validate analytics output
